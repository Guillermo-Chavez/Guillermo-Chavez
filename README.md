## 📂 Proyectos destacados

### 🥇 Predicción de recuperación de oro
Modelo de Machine Learning para predecir la eficiencia de recuperación de oro en dos etapas (*rougher* y *final*) de un proceso industrial de flotación, evaluado con la métrica sMAPE.

- 🛠️ **Tecnologías:** Python · pandas · scikit-learn (Random Forest) · Matplotlib
- 📊 **Resultado clave:** sMAPE de **7.99** en el conjunto de prueba con Random Forest (profundidad=4, n_estimators=100), tras EDA del proceso, imputación de datos y optimización de hiperparámetros
- 💻 [Repositorio](https://github.com/Guillermo-Chavez/Prototipo-de-un-modelo-de-machine-learning)

### 📉 Predicción de cancelación de clientes (Churn)
Modelo de clasificación para identificar clientes con mayor probabilidad de cancelar su servicio, con el fin de dirigir campañas de retención de forma proactiva.

- 🛠️ **Tecnologías:** Python · pandas · scikit-learn (Regresión Logística, Random Forest, Gradient Boosting)
- 📊 **Resultado clave:** mejor modelo Gradient Boosting con **ROC-AUC 0.85**, tras integrar múltiples fuentes de datos, ingeniería de variables (antigüedad del cliente) y ajuste de hiperparámetros con GridSearchCV/RandomizedSearchCV
- 💻 [Repositorio](https://github.com/Guillermo-Chavez/Prediccion-de-cancelacion-de-clientes)

### 🚗 Aplicación web para análisis del mercado automotriz
Aplicación web desarrollada con Streamlit para el análisis exploratorio de datos del mercado de vehículos: precio, tipo de tracción, número de cilindros, transmisión y otras características mediante gráficos dinámicos.

- 🔗 **Demo en vivo:** [web-site-p9yw.onrender.com](https://web-site-p9yw.onrender.com)
- 🛠️ **Tecnologías:** Python · Streamlit · pandas
- 💻 [Repositorio](https://github.com/Guillermo-Chavez/App-de-datos-de-automoviles)

### 🏎️ Analizador de Fórmula 1
Dashboard interactivo desarrollado con Streamlit y FastF1 para explorar datos oficiales de F1: comparación de telemetría entre pilotos vuelta a vuelta (velocidad, acelerador, freno, RPM y marchas), delta de tiempo y mapa del circuito coloreado por velocidad.

- 🔗 **Demo en vivo:** [analizador-de-datos-f1.onrender.com](https://analizador-de-datos-f1.onrender.com)
- 🛠️ **Tecnologías:** Python · Streamlit · FastF1 · Plotly · Matplotlib
- 💻 [Repositorio](https://github.com/Guillermo-Chavez/Analizador-de-datos-f1)

> ⏳ Ambas demos están alojadas en el plan gratuito de Render: si nadie las visitó recientemente pueden tardar 30–60 segundos en "despertar".
