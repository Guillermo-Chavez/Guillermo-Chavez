## 📂 Proyectos destacados

### 🥇 [Predicción de recuperación de oro](https://github.com/Guillermo-Chavez/Prototipo-de-un-modelo-de-machine-learning)
Modelo de Machine Learning para predecir la eficiencia de recuperación de oro en dos etapas (*rougher* y *final*) de un proceso industrial de flotación, evaluado con la métrica sMAPE.

**Tecnologías:** Python · pandas · scikit-learn (Random Forest) · Matplotlib

**Resultados:**
- Análisis exploratorio del proceso industrial
- Limpieza e imputación de datos
- Optimización de hiperparámetros
- Modelo final con buen desempeño utilizando Random Forest

---

### 📉 [Predicción de cancelación de clientes (Churn)](https://github.com/Guillermo-Chavez/Prediccion-de-cancelacion-de-clientes)
Modelo de clasificación para identificar clientes con mayor probabilidad de cancelar su servicio, con el fin de dirigir campañas de retención de forma proactiva.

**Tecnologías:** Python · pandas · scikit-learn (Regresión Logística, Random Forest, Gradient Boosting)

**Resultados:**
- Integración de múltiples fuentes de datos y tratamiento de valores nulos post-integración
- Ingeniería de variables (antigüedad del cliente) y EDA orientado a negocio
- Ajuste de hiperparámetros con GridSearchCV / RandomizedSearchCV
- Mejor modelo: Gradient Boosting — **ROC-AUC 0.85**

---

Modelo de series temporales para pronosticar el consumo de energía utilizando variables históricas y características temporales.
